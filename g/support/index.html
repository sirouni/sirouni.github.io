WOODSLIDE SUPPORT

常见问题 · FAQ

进度会丢失吗？ / Is my progress safe?

关卡进度保存在设备本地，删除 App 会清空进度，更新 App 不受影响。
Progress is stored on your device. Updating the app keeps it; deleting the app clears it.

朋友分享的链接打不开？ / A shared link doesn't open the game?

请确认已安装最新版本，并直接在系统浏览器或消息 App 中点击链接（部分应用的内置浏览器需选择「在默认浏览器中打开」）。
Make sure the latest version is installed and open the link outside in-app browsers.

没有声音？ / No sound?

游戏尊重系统设置，请检查静音开关与媒体音量，以及游戏内设置中的音效开关。
Check the silent switch, media volume, and the in-game sound toggle.

Found a bug or have a suggestion? File an issue on GitHub — we usually reply within 1–2 days.
